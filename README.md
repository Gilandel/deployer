# A simple maven deployer

Required properties (defined in Travis):
- DEPLOYER_URL (ex: https://raw.githubusercontent.com/Gilandel/deployer/master)
- ENCPRYPTED_KEY
- ENCPRYPTED_IV
- GIT_EMAIL
- GIT_USER
- OSSRH_JIRA_USERNAME
- OSSRH_JIRA_PASSWORD
- GROUP_ID_PATH (ex: fr/landel/utils)
